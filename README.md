# Горячие клавиши
#### Копировать: 
ctrl+C
#### Вставить: 
ctrl+V
#### Сохранить: 
ctrl+S
#### Вырезать: 
ctrl+X
#### Отмена: 
ctrl+Z

# Работа с git
### Скачать проект:
```bash
git clone <URL>
```
### Учесть все файлы:
```bash
git add .
```
### Сохранять коммит:
```bash
git commit -m "..."
```
### Отправка в облако:
```bash
git push
```
